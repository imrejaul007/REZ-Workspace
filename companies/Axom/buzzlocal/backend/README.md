# BuzzLocal Backend

This directory is a placeholder. The actual backend implementation for BuzzLocal is integrated into the main buzzlocal service.

## Architecture

The BuzzLocal backend services are located in the parent buzzlocal directory:

```
buzzlocal/
├── buzzlocal-api/           # Main API service
├── buzzlocal-graph/         # Knowledge graph service
├── buzzlocal-intelligence/   # AI intelligence service
└── buzzlocal-push/          # Push notification service
```

## Integration

This `backend/` directory is kept for backward compatibility references only. All backend functionality is handled by the services listed above.
