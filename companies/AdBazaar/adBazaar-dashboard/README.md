# AdBazaar Dashboard

Unified dashboard - single pane of glass for the entire AdBazaar platform.

## Service Purpose

Next.js-based unified dashboard providing a comprehensive view of all AdBazaar services. Displays real-time analytics, campaign management, inventory overview, and billing information.

## Port

```
3016
```

## Getting Started

```bash
# Install dependencies
npm install

# Development mode
npm run dev

# Build for production
npm run build

# Start production server
npm start

# Linting
npm run lint
```

## Tech Stack

- Next.js 14.2
- React 18.3
- TypeScript
- Tailwind CSS 4.3
- Axios (API client)
- Autoprefixer

## Features

- Real-time campaign analytics
- Unified inventory management
- Cross-service performance metrics
- Billing and payment management
- Audience insights dashboard
- Creative preview and management

## Project Structure

```
src/
├── app/           # Next.js App Router pages
├── components/    # Reusable UI components
```
