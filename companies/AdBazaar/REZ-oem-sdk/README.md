# REZ OEM SDK - Android

Pre-installed Android SDK for device partnerships.

## Features

- Lockscreen ads
- Notification ads
- System app integration
- Carrier billing
- Pre-installed apps
- OEM dashboard

## Setup

```gradle
implementation 'com.rezapp:oemsdk:1.0.0'
```

## Usage

```kotlin
REZOEM.init(context)
REZOEM.showAd()
```
