# AdBazaar Analytics Dashboard

Real-time ad campaign analytics for AdBazaar.

## Quick Start

```bash
cd REZ-ads-analytics-dashboard
npm install
npm run dev
```

## Features

- Campaign performance metrics
- CTR, CPC, CPA tracking
- Conversion analytics
- Time range filtering
- Export reports

## Pages

| Route | Description |
|-------|-------------|
| `/` | Main dashboard |
| `/campaigns` | Campaign list |
| `/audiences` | Audience analytics |
| `/billing` | Billing overview |
