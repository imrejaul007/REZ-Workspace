# REZ Chatbot Builder UI

**Version:** 0.1
**Date:** May 2026

---

## WHAT IT DOES

```
Visual Flow Builder → AI Chatbot → WhatsApp/Website Integration
```

### Features

| Feature | Description |
|---------|-------------|
| Visual Flow Builder | Drag-and-drop chatbot flow designer |
| Node Types | Message, Question, Condition, Action, AI |
| AI Integration | GPT-powered responses with context |
| Multi-Channel | WhatsApp and website deployment |
| Analytics | Conversation metrics and insights |

---

## PAGES

| Page | Purpose |
|------|---------|
| `/` | Dashboard with bot list |
| `/builder/[id]` | Visual flow editor |
| `/analytics` | Conversation analytics |
| `/settings` | Bot configuration |
