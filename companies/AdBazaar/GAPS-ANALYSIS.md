## Fix Commands

### Build all services
```bash
for dir in */; do
  if [ -f "$dir/package.json" ]; then
    cd "$dir" && npm run build 2>/dev/null && cd ..
  fi
done
```

### Create Dockerfile for all services
```bash
for dir in */; do
  if [ -f "$dir/package.json" ] && [ ! -f "$dir/Dockerfile" ]; then
    # Copy template
    cp Dockerfile.template "$dir/Dockerfile"
  fi
done
```
