Quick Setup

1
Add the Script
Include the widget script in your HTML
2
Create Container
Add a container element for the widget
3
Initialize
Configure and initialize the widget
<!-- Add the widget container --> <div id="rez-chat"></div> <!-- Load the widget --> <script src="https://cdn.rezapp.com/live-chat-widget/widget.umd.js"></script> <script> window.ReZLiveChat.create('rez-chat', { apiUrl: 'https://api.example.com', apiKey: 'your-api-key', companyName: 'Acme Support', primaryColor: '#6366f1' }); </script>

Configuration Options

Option Type Default Description
apiUrl string - Your chat API endpoint
apiKey string - API key for authentication
companyName string Support Display name for your company
primaryColor string #6366f1 Primary accent color
position string bottom-right Chat bubble position
enableFileUpload boolean true Enable file attachments
enableQuickReplies boolean true Enable quick reply buttons

JavaScript API

// Get widget instance const chat = window.ReZLiveChat.getInstance(); // Listen to events chat.on('message_received', (event) => { console.log('New message:', event.data); }); // Programmatic actions chat.open(); chat.close(); chat.toggle(); // Send message await chat.sendMessage('Hello!'); // Get state const state = chat.getState(); console.log(state.messages, state.agent);

Events

Event Description
readyWidget initialized
openChat window opened
closeChat window closed
message_sentMessage sent successfully
message_receivedNew message received
agent_joinedAgent joined session
agent_leftAgent left session
