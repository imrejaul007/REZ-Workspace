# REZ Merchant WhatsApp Manager

**Version:** 1.0
**Date:** May 2026

---

## WHAT IT DOES

```
WhatsApp Business API → Merchant Dashboard → Customer Engagement
```

### Features

| Feature | Description |
|---------|-------------|
| WhatsApp Integration | Connect WhatsApp Business API |
| Template Management | Create and manage message templates |
| Campaign Management | Send bulk WhatsApp campaigns |
| Analytics | Track message delivery and engagement |
| Auto-Replies | Configure automated responses |
