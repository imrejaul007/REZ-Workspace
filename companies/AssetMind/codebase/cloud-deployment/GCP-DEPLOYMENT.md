# AssetMind GCP Deployment Guide

**Platform:** Google Cloud Platform  
**Date:** June 12, 2026

---

## Overview

Deploy AssetMind on Google Cloud Platform using:
- **Cloud Run** for serverless containers
- **Cloud SQL** for PostgreSQL
- **Memorystore** for Redis
- **Cloud Load Balancing** for traffic
- **Cloud Monitoring** for observability

## Prerequisites

- Google Cloud SDK (`gcloud`)
- Docker
- Project with billing enabled

## Architecture

```
Internet → Cloud Load Balancing → Cloud Run (API Gateway)
                                        ↓
                                Cloud Run (Services)
                                        ↓
                    ┌─────────────────┼─────────────────┐
                    ↓                 ↓                 ↓
              Cloud SQL         Memorystore        Cloud Storage
            PostgreSQL            Redis               Files
```

## Step 1: Setup Project

```bash
# Set project
gcloud config set project YOUR_PROJECT_ID

# Enable APIs
gcloud services enable \
  run.googleapis.com \
  sqladmin.googleapis.com \
  redis.googleapis.com \
  cloudbuild.googleapis.com \
  artifactregistry.googleapis.com
```

## Step 2: Create Cloud SQL PostgreSQL

```bash
# Create instance
gcloud sql instances create assetmind-db \
  --database-version=POSTGRES_15 \
  --tier=db-custom-2-8192 \
  --region=us-central1 \
  --storage-type=SSD \
  --storage-size=50GB \
  --backup-start-time=03:00

# Create database
gcloud sql databases create assetmind --instance=assetmind-db

# Create user
gcloud sql users create assetmind \
  --instance=assetmind-db \
  --password=SECURE_PASSWORD
```

## Step 3: Create Memorystore Redis

```bash
# Create Redis instance
gcloud redis instances create assetmind-redis \
  --size=1 \
  --region=us-central1 \
  --network=default \
  --redis-version=redis_7_0

# Get connection info
gcloud redis instances describe assetmind-redis --region=us-central1
```

## Step 4: Create Artifact Registry

```bash
# Create repository
gcloud artifacts repositories create assetmind \
  --repository-format=docker \
  --location=us-central1

# Configure Docker auth
gcloud auth configure-docker us-central1-docker.pkg.dev
```

## Step 5: Build and Push Images

```bash
# Build API Gateway image
cd assetmind-api-gateway
gcloud builds submit --tag us-central1-docker.pkg.dev/YOUR_PROJECT_ID/assetmind/api-gateway:latest

# Build Twin Engine
cd ../assetmind-twin-engine
gcloud builds submit --tag us-central1-docker.pkg.dev/YOUR_PROJECT_ID/assetmind/twin-engine:latest
```

## Step 6: Deploy to Cloud Run

```bash
# Deploy API Gateway
gcloud run deploy assetmind-api-gateway \
  --image=us-central1-docker.pkg.dev/YOUR_PROJECT_ID/assetmind/api-gateway:latest \
  --platform=managed \
  --region=us-central1 \
  --allow-unauthenticated \
  --min-instances=1 \
  --max-instances=10 \
  --memory=512Mi \
  --cpu=1 \
  --set-env-vars="NODE_ENV=production"
```

## Step 7: Secret Manager

```bash
# Store secrets
echo -n "postgresql://assetmind:PASSWORD@/assetmind?host=/cloudsql/PROJECT:region:instance" | \
  gcloud secrets create DATABASE_URL --data-file=-

gcloud secrets create SECRET_KEY --data-file=- <<< "your-secret-key"
```
