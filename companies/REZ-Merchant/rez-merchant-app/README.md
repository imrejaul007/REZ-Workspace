# rez-merchant-app

**Type:** Frontend/UI Application  
**Platform:** React/React Native

## Overview

This is a frontend UI application. It does not have a standalone backend server.

## Technology Stack

- React 18
- React Native / Vite / Next.js
- TypeScript

## Running Locally

```bash
cd rez-merchant-app
npm install
npm start
```

## API Integration

This UI connects to backend services:
- `rez-merchant-service` (Port 4005)
- `rez-pos-service` (Port 4081)
- Industry-specific services
