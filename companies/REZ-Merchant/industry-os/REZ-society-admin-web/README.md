# REZ Society Admin Portal

**Port:** 3029 (updated from default)  
**Industry:** Residential Society  
**Company:** REZ-Merchant

## Overview

Comprehensive society management dashboard for residential complexes, housing societies, and apartment management.

## Features

- 📊 **Dashboard** - Society overview
- 👥 **Residents** - Resident management
- 🚪 **Visitors** - Visitor management
- 🔧 **Maintenance** - Maintenance requests
- 🔒 **Security** - Security management
- 💰 **Billing** - Society billing

## Quick Start

```bash
cd REZ-society-admin-web
pnpm install
pnpm dev  # → http://localhost:3029
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons
