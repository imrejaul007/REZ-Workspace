# REZ Spa Web

**Port: 4403**

Spa management web application for beauty and wellness businesses.

## Features

- Dashboard with key metrics
- Service management
- Booking calendar
- Therapist scheduling
- Inventory overview

## Pages

- `/` - Dashboard
- `/services` - Service management
- `/bookings` - Booking calendar and management

## Tech Stack

- Next.js 14
- React 18
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```
