# REZ Retail Web - Admin Dashboard

Next.js-based admin dashboard for REZ Retail management.

## Features

- Dashboard with key metrics
- Product management
- Inventory tracking with alerts
- Customer management
- Quick actions

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- TypeScript
- Recharts for data visualization

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/` - Dashboard
- `/products` - Product management
- `/inventory` - Inventory tracking
- `/customers` - Customer management
