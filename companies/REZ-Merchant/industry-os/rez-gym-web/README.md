# REZ Gym Web

REZ Gym OS - Web dashboard for gym management

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS

## Quick Start

```bash
npm install
npm run dev
```

- rez-gym-attendance-service (4301)
