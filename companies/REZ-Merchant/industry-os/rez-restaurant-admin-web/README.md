# REZ Restaurant Admin Web

Admin dashboard for restaurant management.

## Overview

Web-based admin interface for:
- Menu management
- Order tracking
- Staff management
- Analytics dashboard

## Tech Stack

- React 18
- React Router DOM
- Vite

## Features

- Real-time order monitoring
- Menu CRUD operations
- Staff scheduling
- Revenue analytics
- Customer management

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

```
VITE_API_URL=http://localhost:4012
```
