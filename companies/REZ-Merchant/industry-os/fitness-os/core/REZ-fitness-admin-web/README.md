# REZ Fitness Admin Portal

**Port:** 3020  
**Industry:** Fitness & Gym  
**Company:** REZ-Merchant

## Overview

Comprehensive fitness center management dashboard for gyms, yoga studios, and sports centers.

## Features

- 📊 **Dashboard** - Fitness center overview
- 👥 **Members** - Member management
- 🏋️ **Trainers** - Trainer profiles and scheduling
- 📅 **Classes** - Class scheduling and booking
- ✅ **Attendance** - Member check-in/out
- 💰 **Packages** - Membership packages

## Quick Start

```bash
cd REZ-fitness-admin-web
pnpm install
pnpm dev  # → http://localhost:3020
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons
