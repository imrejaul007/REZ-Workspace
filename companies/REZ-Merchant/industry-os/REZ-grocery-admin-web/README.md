# REZ Grocery Admin Portal

**Port:** 3011  
**Industry:** Grocery & Retail  
**Company:** REZ-Merchant

## Overview

Comprehensive grocery store management dashboard for supermarkets, kirana stores, and quick commerce platforms.

## Features

- 📊 **Dashboard** - Store overview
- 📦 **Products** - Product catalog management
- 🏪 **Inventory** - Stock tracking
- 🚚 **Suppliers** - Supplier management
- 📍 **Delivery** - Delivery tracking
- ⚠️ **Expiry Alerts** - Stock expiry notifications

## Quick Start

```bash
cd REZ-grocery-admin-web
pnpm install
pnpm dev  # → http://localhost:3011
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons
