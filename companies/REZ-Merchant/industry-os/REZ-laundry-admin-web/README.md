# REZ Laundry Admin Portal

**Port:** 3026 (updated from default)  
**Industry:** Laundry & Dry Clean  
**Company:** REZ-Merchant

## Overview

Comprehensive laundry management dashboard for dry cleaners, laundromats, and laundry service providers.

## Features

- 📊 **Dashboard** - Operations overview
- 📋 **Orders** - Order management
- 👕 **Garments** - Garment tracking
- 💰 **Billing** - Invoicing and payments
- 👥 **Customers** - Customer management
- 📍 **Pickup/Delivery** - Service scheduling

## Quick Start

```bash
cd REZ-laundry-admin-web
pnpm install
pnpm dev  # → http://localhost:3026
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons
