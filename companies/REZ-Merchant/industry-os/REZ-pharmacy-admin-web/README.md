# REZ Pharmacy Admin Portal

**Port:** 3031 (updated from 3013)  
**Industry:** Pharmacy  
**Company:** REZ-Merchant

## Overview

Comprehensive pharmacy management dashboard for drugstores, medical shops, and pharmacy chains.

## Features

- 📊 **Dashboard** - Pharmacy overview
- 💊 **Medicines** - Medicine catalog
- 📝 **Prescriptions** - Prescription management
- 📦 **Inventory** - Stock management
- ⚠️ **Expiry Alerts** - Medicine expiry tracking
- 📍 **Suppliers** - Supplier management

## Quick Start

```bash
cd REZ-pharmacy-admin-web
pnpm install
pnpm dev  # → http://localhost:3031
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons
