# REZ Gym Attendance Service

REZ Gym OS - Attendance tracking and QR-based access control

**Port:** 4301

## Features

- **QR Code Access**: Generate and validate QR codes for gym entry
- **Attendance Tracking**: Track member check-ins and check-outs
- **Session Management**: Manage gym access sessions
- **Real-time Tracking**: Live gym occupancy monitoring
- **Access Logs**: Complete audit trail of gym entries

## Quick Start

```bash
npm install
npm run dev
```
