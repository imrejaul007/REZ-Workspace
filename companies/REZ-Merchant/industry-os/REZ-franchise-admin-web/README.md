# REZ Franchise Admin Portal

**Port:** 3021  
**Industry:** Franchise Management  
**Company:** REZ-Merchant

## Overview

Comprehensive franchise management dashboard for franchise networks, multi-location businesses, and chain operators.

## Features

- 📊 **Dashboard** - Network overview
- 📍 **Locations** - Multi-location management
- 💰 **Royalties** - Royalty tracking
- 📈 **Performance** - Location comparison
- 📦 **Inventory** - Central inventory
- ✅ **Compliance** - Policy compliance tracking

## Quick Start

```bash
cd REZ-franchise-admin-web
pnpm install
pnpm dev  # → http://localhost:3021
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons
