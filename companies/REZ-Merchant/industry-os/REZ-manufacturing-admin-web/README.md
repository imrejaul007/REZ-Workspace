# REZ Manufacturing Admin Portal

**Port:** 3027 (updated from default)  
**Industry:** Manufacturing  
**Company:** REZ-Merchant

## Overview

Comprehensive manufacturing management dashboard for production planning, quality control, and operations.

## Features

- 📊 **Dashboard** - Production overview
- 🏭 **Production** - Production planning
- 📋 **BOM** - Bill of materials
- ✅ **Quality** - Quality control
- 📦 **Inventory** - Raw materials and finished goods
- ⚙️ **Machines** - Machine monitoring

## Quick Start

```bash
cd REZ-manufacturing-admin-web
pnpm install
pnpm dev  # → http://localhost:3027
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons
