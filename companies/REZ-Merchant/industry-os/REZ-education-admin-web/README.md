# REZ Education Admin Portal

**Port:** 3009  
**Industry:** Education  
**Company:** REZ-Merchant

## Overview

Comprehensive education management dashboard for schools, colleges, coaching centers, and training institutions.

## Features

- 📊 **Dashboard** - Academic overview
- 👥 **Students** - Student enrollment and management
- 📚 **Courses** - Course and curriculum management
- ✅ **Attendance** - Attendance tracking
- 📝 **Results** - Examination and grade management
- 💰 **Fees** - Fee collection and management

## Quick Start

```bash
cd REZ-education-admin-web
pnpm install
pnpm dev  # → http://localhost:3009
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons
