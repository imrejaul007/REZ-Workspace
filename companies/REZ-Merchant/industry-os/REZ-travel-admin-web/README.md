# Travel Admin Portal

**Location:** `/REZ-Merchant/industry-os/REZ-travel-admin-web/`
**Tech Stack:** Next.js 14 + React 18 + Tailwind CSS
**Port:** 3002

## Features
- Dashboard with booking stats
- Flight management
- Hotel listings
- Destination management
- Customer management
- Package deals

## Quick Start
```bash
cd REZ-travel-admin-web
npm install
npm run dev
```

- KHAIRMOVE (Airzy)
