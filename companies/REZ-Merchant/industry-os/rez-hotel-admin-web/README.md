# Hotel Admin Web Panel

A modern hotel management dashboard for managers and owners built with Next.js 15, React 19, and Tailwind CSS.

## Features

- **Key Metrics Dashboard** - Real-time overview of revenue, occupancy, ratings, and more
- **Recent Bookings Table** - Manage and track guest reservations
- **Room Status** - Monitor room availability and maintenance status
- **Staff On Duty** - Track personnel across departments
- **Quick Actions** - Fast access to common management tasks

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the panel.

## Target Users

- Hotel Managers
- Hotel Owners
- Front Desk Supervisors

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
