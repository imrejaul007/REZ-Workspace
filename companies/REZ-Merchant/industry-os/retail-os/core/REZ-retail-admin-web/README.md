# REZ Retail Admin Portal

**Port:** 3015  
**Industry:** Retail  
**Company:** REZ-Merchant

## Overview

Comprehensive retail management dashboard for shops, malls, and retail chains.

## Features

- 📊 **Dashboard** - Store overview
- 📦 **Products** - Product catalog
- 🏪 **Inventory** - Stock management
- 💳 **POS** - Point of sale
- 🎁 **Loyalty** - Loyalty program
- 🚚 **Suppliers** - Supplier management

## Quick Start

```bash
cd REZ-retail-admin-web
pnpm install
pnpm dev  # → http://localhost:3015
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons
