# REZ Salon Admin Web

Admin dashboard for salon management.

## Overview

Web-based admin interface for:
- Appointment management
- Service catalog
- Staff scheduling
- Customer loyalty

## Tech Stack

- React 18
- React Router DOM
- Vite

## Features

- Appointment calendar
- Service management
- Staff performance tracking
- Inventory management
- Marketing tools

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

```
VITE_API_URL=http://localhost:4010
```
