# REZ Real Estate Admin Portal

**Port:** 3028 (updated from default)  
**Industry:** Real Estate  
**Company:** REZ-Merchant

## Overview

Comprehensive real estate management dashboard for property developers, brokers, and agents.

## Features

- 📊 **Dashboard** - Sales overview
- 🏠 **Properties** - Property listings
- 👥 **Leads** - Lead management
- 📅 **Site Visits** - Visit scheduling
- 🤝 **Deals** - Deal pipeline
- 💰 **Commissions** - Commission tracking

## Quick Start

```bash
cd REZ-real-estate-admin-web
pnpm install
pnpm dev  # → http://localhost:3028
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons
