# REZ Salon Web

REZ Salon OS - Web dashboard for salon management

## Features

- **Dashboard**: Overview of salon operations
- **Booking Management**: View and manage appointments
- **Customer Management**: View customer list and details
- **Service Management**: Manage salon services
- **Staff Management**: View and manage stylists
- **Calendar View**: Visual booking calendar
- **Analytics**: Salon performance insights

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Services Integration

- rez-salon-service (4200) - Core services
- rez-salon-booking-service (4201) - Bookings
- rez-salon-crm-service (4204) - CRM
