# REZ Spa Admin Portal

**Port:** 3030 (updated from 3012)  
**Industry:** Spa & Wellness  
**Company:** REZ-Merchant

## Overview

Comprehensive spa and wellness management dashboard for spas, salons, and wellness centers.

## Features

- 📊 **Dashboard** - Operations overview
- 📅 **Appointments** - Booking management
- 👥 **Clients** - Client management
- 💆 **Services** - Service catalog
- 💰 **Billing** - Invoicing
- 📦 **Products** - Product inventory

## Quick Start

```bash
cd REZ-spa-admin-web
pnpm install
pnpm dev  # → http://localhost:3030
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons
