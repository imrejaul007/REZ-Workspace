# REZ Auto Admin Portal

**Port:** 3023  
**Industry:** Automotive  
**Company:** REZ-Merchant

## Overview

Comprehensive automotive service management dashboard for vehicle service centers, showrooms, and auto repair shops.

## Features

- 🚗 **Dashboard** - Service overview and metrics
- 🔧 **Service Records** - Complete vehicle service history
- 👥 **Customers** - Customer management
- 📦 **Inventory** - Parts and supplies tracking
- 📋 **Job Cards** - Service job management
- 📊 **Reports** - Service analytics

## Quick Start

```bash
cd REZ-auto-admin-web
pnpm install
pnpm dev  # → http://localhost:3023
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- Lucide Icons
