# REZ Atlas Dashboard

Enterprise Intelligence Dashboard for REZ Atlas - The Merchant Intelligence Network for the Physical World.

## Features

- **Acquisition Dashboard** - Track new merchants, leads, and conversion funnel
- **Territory Dashboard** - View coverage, visits, and performance by territory
- **Opportunity Dashboard** - Analyze revenue potential and upsell opportunities
- **Merchant Health Dashboard** - Monitor risk, growth, and satisfaction metrics

## Tech Stack

- Next.js 14
- React 18
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Dashboard will be available at: http://localhost:5190
