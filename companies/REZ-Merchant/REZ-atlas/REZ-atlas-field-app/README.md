# REZ Atlas Field App

Mobile app for field sales teams to manage routes, visits, and merchant interactions.

## Features

- **Route View** - View today's route with all stops
- **Navigation** - Turn-by-turn directions to merchant locations
- **Visit Logging** - Log visit outcomes and notes
- **Merchant Info** - View merchant twin data and recommendations
- **Offline Mode** - Work without internet connection

## Tech Stack

- React Native / Expo
- TypeScript

## Getting Started

```bash
npm install
npm run dev
# or
npm run android
npm run ios
```

App connects to REZ Atlas Gateway at: http://localhost:5150
