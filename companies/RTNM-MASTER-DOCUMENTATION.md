## 🚀 Quick Start

### Start AssetMind Platform
```bash
cd AssetMind/codebase
./START.sh docker  # Start with Docker
./START.sh local   # Start locally
./integration-tests.sh  # Run tests
```

### Deploy to Cloud
```bash
# AWS
cat cloud-deployment/AWS-DEPLOYMENT.md

# GCP
cat cloud-deployment/GCP-DEPLOYMENT.md

# Azure
cat cloud-deployment/AZURE-DEPLOYMENT.md
```

### Build Mobile App
```bash
cd assetmind-mobile
eas build --platform ios --profile production
eas build --platform android --profile production
```
