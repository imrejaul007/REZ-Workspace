# rez-driver

**Driver Partner App**

## Status

✅ COMPLETE

## Overview

Delivery and ride partner application for REZ ecosystem.

## Features

- Deliveries management
- Rides management
- Earnings tracking
- Profile management
- Driver settings

## Screens

- Deliveries (Tab)
- Rides (Tab)
- Earnings (Tab)
- Profile (Tab)
- Settings (Tab)

## Tech Stack

- Expo SDK 52
- React Native
- TypeScript
- Zustand

## Quick Start

```bash
npm install
npx expo start
```

## Backend

Connects to delivery service on port `4010`

## API Services

| Service | Description |
|---------|-------------|
| `driverApi` | Driver profile |
| `deliveryApi` | Deliveries & requests |
| `notificationsApi` | Push notifications |
