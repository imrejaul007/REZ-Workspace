## Go-to-Market Strategy

### Phase 1: ReZ User Migration

```
Strategy: Convert existing ReZ users to Do users

Leverage:
• ReZ's existing user base (organic)
• ReZ's merchant network (supply)
• ReZ's payment infrastructure (trust)

Tactic: "Try Do — the new way to use ReZ"
```

### Phase 2: New User Acquisition

```
Strategy: Position Do as the AI assistant for daily life

Channels:
• Social media (Instagram, Twitter)
• Influencer partnerships
• App Store optimization
• Word of mouth

Tactic: "Your AI that actually does" → demo videos
```

### Phase 3: Merchant Expansion

```
Strategy: Onboard merchants to Do's platform

Incentives:
• Lower fees for AI-booked customers
• Featured in Do's discovery
• Karma program benefits

Tactic: "Get more bookings via Do"
```
