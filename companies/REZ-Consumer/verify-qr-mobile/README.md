# verify-qr-mobile

**Product Verification Mobile App**

## Overview

QR code scanning and product warranty verification for consumers.

## Status

✅ COMPLETE - Built June 2026

## Features

- QR code scanning
- Serial number verification
- Warranty check
- Claims management
- Service center locator

## Screens

- Home
- Scan QR
- Verify by Serial
- Warranty Check
- Claims List
- Profile
- Service Centers

## Tech Stack

- Expo SDK 53
- React Native
- TypeScript
- React Query
- Zustand

## Quick Start

```bash
npm install
npx expo start
```

## API Integration

Connects to `verify-qr-service` on port `4003`
