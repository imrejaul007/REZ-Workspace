# REZ WooCommerce Connector

Sync products and orders between REZ and WooCommerce.

## Quick Start

```bash
npm install
npm run dev
```

## Environment Variables

```bash
WOOCOMMERCE_URL=https://yourstore.com
WOOCOMMERCE_KEY=ck_xxx
WOOCOMMERCE_SECRET=cs_xxx
```

## Features

- Product sync
- Order sync
- Inventory sync
- Webhook support
