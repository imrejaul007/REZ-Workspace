**Link-in-bio for the modern creator.**

## Features

| Feature | Description |
|---------|-------------|
| **Bio Links** | All your links in one place |
| **Themes** | 50+ beautiful themes |
| **Custom Domain** | Use your own domain |
| **Analytics** | Click & visit tracking |
| **Scheduling** | Time-based visibility |
| **QR Codes** | Auto-generated QR |
| **Password Protection** | Per-link or full page |
| **AI Themes** | Generate custom themes |

---

## Quick Start

```bash
cd RABTUL-Technologies/REZ-links-service
npm install
npm run dev
```

**Port:** 4101

---

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/links` | List link pages |
| POST | `/api/links` | Create link page |
| GET | `/api/links/:username` | Get page |
| PATCH | `/api/links/:username` | Update page |
| DELETE | `/api/links/:username` | Delete page |
| POST | `/api/links/:username/links` | Add link |
| GET | `/api/links/:username/analytics` | Get analytics |
