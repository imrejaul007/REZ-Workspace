# REZ Unified Loyalty

Cross-platform loyalty points and rewards.

## Quick Start

```bash
npm install
npm run dev
```

## Tiers

- Bronze
- Silver
- Gold
- Platinum

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/earn | Earn points |
| POST | /api/redeem | Redeem points |
| GET | /api/balance/:userId | Get balance |
