---

## Quick Start

```bash
cd RABTUL-Technologies/REZ-pages-service
npm install
npm run dev
```

**Port:** 4100

---

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/pages` | List pages |
| POST | `/api/pages` | Create page |
| GET | `/api/pages/:slug` | Get page |
| PATCH | `/api/pages/:slug` | Update page |
| DELETE | `/api/pages/:slug` | Delete page |
| GET | `/:slug` | Public page |
