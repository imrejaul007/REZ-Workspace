# REZ Privacy Layer

PII masking, consent management, and data anonymization.

## Quick Start

```bash
npm install
npm run dev
```

## Features

- Email/phone masking
- Consent management
- Data anonymization
- GDPR compliance

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/mask | Mask PII data |
| POST | /api/consent | Manage consent |
| POST | /api/anonymize | Anonymize data |
