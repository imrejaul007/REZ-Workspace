# Hospitality OS

> Guest Services Platform

**Port:** 5040 | **Status:** ✅ Production Ready

## Overview

Comprehensive guest services platform for hotels, resorts, and hospitality businesses with AI-powered experiences.

## Quick Start

```bash
cd industries/hospitality-os
npm install
node src/index.js
```

Access at: http://localhost:5040

## Digital Twins

- **Guest Twin** - Guest profiles, preferences
- **Room Twin** - Room inventory, status
- **Booking Twin** - Reservations, history
- **Revenue Twin** - Financial analytics
- **Service Twin** - Amenities, requests

## AI Agents

- Concierge, Booking, Upsell, Housekeeping, Revenue, Feedback
