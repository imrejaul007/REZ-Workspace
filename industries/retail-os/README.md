# Retail OS

> Commerce Platform

**Port:** 5030 | **Status:** ✅ Production Ready

## Overview

Unified commerce platform connecting customers, products, inventory, orders, and revenue analytics with AI-powered shopping experiences.

## Quick Start

```bash
cd industries/retail-os
npm install
node src/index.js
```

Access at: http://localhost:5030

## Digital Twins

- **Customer Twin** - Shopping behavior, preferences
- **Product Twin** - Product catalog, pricing
- **Inventory Twin** - Stock management
- **Order Twin** - Transaction tracking
- **Revenue Twin** - Financial analytics

## AI Agents

- Shopping, Checkout, Loyalty, Inventory, VisualSearch, Recommendation
