# NonProfit OS

> Nonprofit Platform

**Port:** 5160 | **Status:** ✅ Production Ready

## Overview

Platform for nonprofit organizations connecting donors, campaigns, beneficiaries, and volunteers with AI-powered fundraising.

## Quick Start

```bash
cd industries/nonprofit-os
npm install
node src/index.js
```

Access at: http://localhost:5160

## Digital Twins

- **Donor Twin** - Donor profiles
- **Campaign Twin** - Fundraising campaigns
- **Beneficiary Twin** - Program recipients
- **Volunteer Twin** - Volunteer management

## AI Agents

- Fundraising, VolunteerMatch, ImpactReport, DonorRel, GrantWriter
