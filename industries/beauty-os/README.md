# Beauty OS

> Beauty & Wellness Platform

**Port:** 5090 | **Status:** ✅ Production Ready

## Overview

Platform for salons, spas, and beauty businesses connecting clients, services, staff, and inventory with AI-powered booking.

## Quick Start

```bash
cd industries/beauty-os
npm install
node src/index.js
```

Access at: http://localhost:5090

## Digital Twins

- **Client Twin** - Client profiles, history
- **Service Twin** - Service catalog
- **Staff Twin** - Stylist management
- **Inventory Twin** - Product inventory

## AI Agents

- Booking, Consultation, ProductRec, Reminder, Satisfaction
