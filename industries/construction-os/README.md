# Construction OS

> Construction Platform

**Port:** 5210 | **Status:** ✅ Production Ready

## Overview

Platform for construction companies connecting projects, contractors, workers, and materials with AI-powered project management.

## Quick Start

```bash
cd industries/construction-os
npm install
node src/index.js
```

Access at: http://localhost:5210

## Digital Twins

- **Project Twin** - Project tracking
- **Contractor Twin** - Contractor management
- **Worker Twin** - Worker profiles
- **Material Twin** - Materials tracking

## AI Agents

- ProjectMgmt, SafetyInsp, ResourceAlloc, ProgressTrack, CostEst
