# Manufacturing OS

> Production Platform

**Port:** 5150 | **Status:** ✅ Production Ready

## Overview

Platform for manufacturing operations connecting products, machines, production lines, and inventory with AI-powered optimization.

## Quick Start

```bash
cd industries/manufacturing-os
npm install
node src/index.js
```

Access at: http://localhost:5150

## Digital Twins

- **Product Twin** - Product definitions
- **Machine Twin** - Equipment tracking
- **Production Twin** - Production planning
- **Inventory Twin** - Raw materials, WIP

## AI Agents

- ProductionSched, QualityCtrl, MaintenancePred, SupplyChain, SafetyInsp
