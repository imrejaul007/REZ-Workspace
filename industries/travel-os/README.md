# Travel OS

> Travel & Tourism Platform

**Port:** 5190 | **Status:** ✅ Production Ready

## Overview

Platform for travel agencies and tourism businesses connecting destinations, packages, bookings, and travelers with AI-powered trip planning.

## Quick Start

```bash
cd industries/travel-os
npm install
node src/index.js
```

Access at: http://localhost:5190

## Digital Twins

- **Destination Twin** - Destination info
- **Package Twin** - Travel packages
- **Booking Twin** - Reservations
- **Traveler Twin** - Traveler profiles

## AI Agents

- TripPlanner, Booking, Concierge, ExpenseTrack, TravelPolicy
