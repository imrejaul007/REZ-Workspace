# Financial OS

> Financial Services Platform

**Port:** 5220 | **Status:** ✅ Production Ready

## Overview

Platform for financial institutions connecting accounts, transactions, customers, and loans with AI-powered fraud detection.

## Quick Start

```bash
cd industries/financial-os
npm install
node src/index.js
```

Access at: http://localhost:5220

## Digital Twins

- **Account Twin** - Account management
- **Transaction Twin** - Transaction tracking
- **Customer Twin** - Customer profiles
- **Loan Twin** - Loan management

## AI Agents

- FraudDetect, CreditAssess, InvestmentAdvisor, ComplianceCheck, KYC
