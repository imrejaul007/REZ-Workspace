# Healthcare OS

> Medical Services Platform

**Port:** 5020 | **Status:** ✅ Production Ready

## Overview

Comprehensive platform for medical services connecting patients, doctors, appointments, billing, and inventory with AI-powered automation.

## Quick Start

```bash
cd industries/healthcare-os
npm install
node src/index.js
```

Access at: http://localhost:5020

## Digital Twins

- **Patient Twin** - HIPAA-compliant records
- **Appointment Twin** - Scheduling and visits
- **Doctor Twin** - Physician management
- **Billing Twin** - Financial operations
- **Inventory Twin** - Medical supplies

## AI Agents

- Intake, Triage, Scheduling, Followup, Prescription, Claims
