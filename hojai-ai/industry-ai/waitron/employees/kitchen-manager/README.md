# Kitchen Manager - Restaurant AI Employee

**Product:** WAITRON
**Type:** AI Agent
**Purpose:** Kitchen coordination, quality control, prep management

## Overview

Kitchen Manager handles kitchen operations and coordinates with the KDS (Kitchen Display System).

## Features

- Order prioritization
- Prep time estimation
- Quality control
- Inventory alerts
- Staff coordination

## API Endpoints

```typescript
GET  /api/ai/kitchen/dashboard  // Kitchen dashboard
POST /api/ai/kitchen/priority    // Set order priority
GET  /api/ai/kitchen/alerts     // Get alerts
```
