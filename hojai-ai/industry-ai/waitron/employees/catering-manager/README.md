# Catering Manager - Restaurant AI Employee

**Product:** WAITRON
**Type:** AI Agent
**Purpose:** Event catering, bulk orders, quotes

## Overview

Catering Manager handles corporate events, weddings, and bulk orders.

## Features

- Event type categorization
- Guest count management
- Menu customization
- Pricing calculation
- Quote generation

## API Endpoints

```typescript
POST /api/ai/catering/quote  // Generate catering quote
POST /api/ai/catering/event   // Book catering event
GET  /api/ai/catering/menu   // Get catering menu
```
