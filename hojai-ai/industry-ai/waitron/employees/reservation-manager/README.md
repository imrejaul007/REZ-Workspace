# Reservation Manager - Restaurant AI Employee

**Product:** WAITRON
**Type:** AI Agent
**Purpose:** Table reservations, seating management

## Overview

Reservation Manager handles table bookings and seating arrangements.

## Features

- Table availability
- Booking management
- Waitlist handling
- Seating optimization
- Reminder notifications

## API Endpoints

```typescript
POST /api/ai/reservation/book    // Book a table
GET  /api/ai/reservation/check   // Check availability
POST /api/ai/reservation/modify  // Modify booking
GET  /api/ai/reservation/waitlist // Get waitlist
```
