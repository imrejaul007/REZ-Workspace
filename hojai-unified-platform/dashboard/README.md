# Hojai Unified Dashboard

**Frontend UI for the Hojai Unified Platform**

## Quick Start

```bash
cd dashboard
npm install
NEXT_PUBLIC_API_URL=http://localhost:4850 npm run dev
```

## Features

- **Real-time Inbox** - Live conversations
- **Multi-channel** - WhatsApp, Instagram, Email
- **Analytics Dashboard** - Metrics and KPIs
- **Campaign Manager** - Create and track campaigns
- **Agent Management** - Team and skills

## Pages

| Route | Description |
|-------|-------------|
| `/` | Main dashboard with inbox |
| `/conversations` | Conversation list |
| `/campaigns` | Campaign manager |
| `/analytics` | Analytics dashboard |
| `/settings` | Settings and configuration |
